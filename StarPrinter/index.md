## <span style="color:red">採用 S3 Browser 連線，請先洽補教雲(5iedu)申請S3帳號與密碼</span>.

### 1. 下載 S3 Browser 下載網址: https://s3browser.com/
![image info](img/Image008.png)
### 2. 安裝 S3 Browser 
### 3. 安裝完成後啟動 S3 Browser --> [Accounts] --> [Add new account...]
![image info](img/Image001.png)
### 4. 新增操作帳號
#### (1) Display name: 輸入用以識別的顯示名稱
#### (2) Account type: 選擇  S3 Compatible Storage 
#### (3) REST Endpoint: 輸入 s3.hicloud.net.tw
#### (4) Access Key ID: 輸入 [補教雲提供之AccessKey]
#### (5) Secret Access Key: 輸入 [補教雲提供之SecretKey]
#### (6) 點選 [Add new account]
![image info](img/Image003.png)
### 5. 增加 External Bucket
#### &emsp;&emsp;按下 [Add new account] 會跳出以下視窗，按下 [是] 
![image info](img/Image005.png)
#### &emsp;&emsp;出現Add External Bucket 請在 Bucket name: 輸入 [補教雲提供之BucketName]，點選 [Add External bucket]
![image info](img/step5-2_20210909-01.png)
### 6. 完成後即可透過 S3 Browser 進行檔案上傳與刪除等管理。
![image info](img/Image010.png)
